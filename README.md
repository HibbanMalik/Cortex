# 🧠 CORTEX - AI Voice Assistant

CORTEX is a beginner-friendly AI voice assistant built in Python.  
It can listen to voice commands, speak responses, and perform basic tasks like opening websites and telling the time.

---

## 🚀 Features

- 🎤 Voice input (speech recognition)
- 🔊 Voice output (text-to-speech)
- 🧠 Wake word support ("cortex")
- 🌐 Open websites (Google, YouTube, SFSU)
- ⏰ Tell current time
- ⌨️ Text mode + Voice mode

---

## 🛠️ Technologies Used

- Python
- pyttsx3 (text-to-speech)
- SpeechRecognition (voice input)
- PyAudio (microphone support)
